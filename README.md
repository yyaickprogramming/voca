# Voca

Веб-сервис для генерации идей стартапов и SaaS-проектов, сохранения лучших вариантов, ведения задач и отслеживания прогресса.

Текущая версия сервера рассчитана на PostgreSQL.

## Локальный запуск

```bash
npm install
npm start
```

По умолчанию сервис запускается на `http://localhost:3000`.
Переменные из локального файла `.env` подхватываются автоматически.

## Переменные окружения

Смотри [.env.example](./.env.example).

Основные:

- `PORT` — порт сервера
- `SESSION_SECRET` — секрет для сессий
- `SESSION_TABLE_NAME` — имя PostgreSQL-таблицы для сессий
- `SESSION_PRUNE_INTERVAL_SECONDS` — как часто чистить просроченные сессии
- `SITE_URL` — публичный адрес сайта
- `DATABASE_URL` — строка подключения к PostgreSQL
- `DATABASE_SSL` — `true`, если подключение требует SSL

## Перед загрузкой в GitHub

- не загружай `.env`, он уже добавлен в `.gitignore`
- не загружай `node_modules`
- не загружай старую локальную SQLite-базу из папки `database`
- в репозитории должны остаться код, `package.json`, `package-lock.json`, `render.yaml`, `.env.example`

## Готовность к деплою

В проект добавлены:

- `render.yaml` для запуска на Render
- `health`-маршрут: `/health`
- `robots.txt` и `sitemap.xml`
- поддержка публичного URL через `SITE_URL`
- поддержка PostgreSQL через `DATABASE_URL`

## Деплой на Render

1. Загрузить проект в GitHub.
2. В Render создать Blueprint из файла `render.yaml`.
3. Render автоматически поднимет web service и Postgres базу.
4. После первого деплоя открыть настройки сервиса и указать `SITE_URL` как публичный адрес сайта.
5. Открыть сайт и проверить маршруты:
   - `/health`
   - `/robots.txt`
   - `/sitemap.xml`

## Важно для роста проекта

Текущая версия использует PostgreSQL не только для данных приложения, но и для хранения пользовательских сессий. Это означает, что вход в аккаунт не пропадет просто из-за перезапуска одного процесса сервера.

Для совсем большого масштаба следующим шагом все еще может стать Redis-совместимое хранилище для сессий, но текущая схема с PostgreSQL уже годится для полноценного MVP и первого продакшен-деплоя.
